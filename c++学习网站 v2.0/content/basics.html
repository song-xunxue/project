C++基础语法

C++是一种通用的、静态类型的编程语言，它支持过程化编程、面向对象编程和泛型编程。本章将详细介绍C++的基础语法。

C++简介

C++是由Bjarne Stroustrup于1979年在贝尔实验室开始开发的一种编程语言，最初被称为"C with Classes"（带类的C）。它既保持了C语言的高效性和灵活性，又增加了面向对象编程的特性。

C++被广泛应用于系统软件、游戏开发、嵌入式系统、高性能计算、金融系统等领域。许多著名的软件和系统都是用C++编写的，包括Windows操作系统、Adobe系列软件、Chrome浏览器和Unreal游戏引擎等。

第一个C++程序

以下是一个简单的C++程序，它输出"Hello, World!"到屏幕上：

#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}

让我们逐行解释这个程序：

  • #include <iostream>: 包含输入输出流库，提供了输入输出功能
  • int main(): 主函数，程序的入口点，每个C++程序都必须有且只有一个main函数
  • std::cout << "Hello, World!" << std::endl;: 输出字符串到屏幕，std::endl表示换行
  • return 0;: 返回0，表示程序正常结束

数据类型

C++是一种静态类型语言，这意味着所有变量在使用前必须声明其类型。C++提供了丰富的数据类型：

基本数据类型

类型 描述 大小(字节) 示例
int 整数类型 4 int a = 10;
float 单精度浮点数 4 float b = 3.14f;
double 双精度浮点数 8 double c = 3.14159;
char 字符类型 1 char d = 'A';
bool 布尔类型 1 bool e = true;
void 无类型 N/A 用于函数无返回值

类型修饰符

C++允许使用修饰符来改变基本数据类型的含义：

  • signed：表示有符号数（默认）
  • unsigned：表示无符号数
  • short：短整型
  • long：长整型
short int si;          // 短整型
long int li;           // 长整型
unsigned int ui;       // 无符号整型
signed long int sli;   // 有符号长整型

变量和常量

变量是程序中用于存储数据的内存位置，而常量是固定值，在程序执行期间不能改变。

变量声明和初始化

// 声明变量
int age;
float salary;

// 初始化变量
age = 25;
salary = 5000.50f;

// 声明并初始化
int score = 95;
std::string name = "John";

// C++11引入的统一初始化语法
int count{10};
double price{19.99};

常量

// 使用const关键字（推荐）
const int MAX_AGE = 100;
const double PI = 3.1415926535;

// 使用constexpr（C++11引入，编译期常量）
constexpr int ARRAY_SIZE = 100;

// 使用#define预处理指令（C风格，不推荐）
#define MIN_AGE 18

作用域

C++变量有不同的作用域：

  • 局部变量：在函数或代码块内部声明，只在该区域内有效
  • 全局变量：在所有函数外部声明，在整个程序中有效
  • 形式参数：在函数参数中声明，作为该函数的局部变量

  • 比较运算符: ==, !=, >, <,>=, <=< /li>

  • 指针运算符: & (取地址), * (解引用)

控制流

C++提供了多种控制流语句来控制程序的执行流程：

条件语句

// if语句
if (age >= 18) {
    std::cout << "成年人" << std::endl;
} else if (age >= 13) {
    std::cout << "青少年" << std::endl;
} else {
    std::cout << "儿童" << std::endl;
}

// switch语句
switch (grade) {
    case 'A':
        std::cout << "优秀" << std::endl;
        break;
    case 'B':
        std::cout << "良好" << std::endl;
        break;
    case 'C':
        std::cout << "及格" << std::endl;
        break;
    default:
        std::cout << "不及格" << std::endl;
}

循环语句

// for循环
for (int i = 0; i < 10; i++) {
    std::cout << i << std::endl;
}

// 范围for循环 (C++11)
int arr[] = {1, 2, 3, 4, 5};
for (int num : arr) {
    std::cout << num << std::endl;
}

// while循环
int j = 0;
while (j < 10) {
    std::cout << j << std::endl;
    j++;
}

// do-while循环
int k = 0;
do {
    std::cout << k << std::endl;
    k++;
} while (k < 10);

跳转语句

  • break: 跳出循环或switch语句
  • continue: 跳过当前循环的剩余部分，进入下一次循环
  • goto: 无条件跳转到标记的语句（不推荐使用）
  • return: 从函数返回

函数

函数是一组一起执行任务的语句，每个C++程序至少有一个main函数。

函数定义

// 函数声明
int add(int a, int b);

// 函数定义
int add(int a, int b) {
    return a + b;
}

// 调用函数
int result = add(5, 3);  // result = 8

函数参数传递

C++支持三种参数传递方式：

  • 传值：函数接收参数的副本，修改不影响原值
  • 传引用：函数接收参数的引用，修改会影响原值
  • 传指针：函数接收参数的指针，通过指针修改原值
// 传值
void incrementByValue(int x) {
    x = x + 1;  // 不影响原值
}

// 传引用
void incrementByReference(int &x) {
    x = x + 1;  // 影响原值
}

// 传指针
void incrementByPointer(int *x) {
    *x = *x + 1;  // 影响原值
}

函数重载

C++允许在同一作用域内声明多个同名函数，只要它们的参数列表不同：

int add(int a, int b) {
    return a + b;
}

double add(double a, double b) {
    return a + b;
}

std::string add(std::string a, std::string b) {
    return a + b;
}

数组

数组是相同类型元素的集合，存储在连续的内存位置中。

数组声明和初始化

// 声明数组
int numbers[5];  // 包含5个整数的数组

// 初始化数组
int primes[5] = {2, 3, 5, 7, 11};

// 省略大小
int primes[] = {2, 3, 5, 7, 11};  // 编译器自动计算大小

// 访问数组元素
int first = primes[0];  // 第一个元素
int last = primes[4];   // 最后一个元素

// 多维数组
int matrix[3][3] = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

标准库数组 (C++11)

C++11引入了std::array，比传统数组更安全：

#include <array>

std::array<int, 5> primes = {2, 3, 5, 7, 11};

// 访问元素
int first = primes[0];
int size = primes.size();  // 获取数组大小

指针

指针是存储变量内存地址的变量，是C++的重要特性。

指针基础

int var = 20;   // 实际变量
int *ptr;       // 指针变量

ptr = &var;     // 在指针变量中存储var的地址

std::cout << "var的值: " << var << std::endl;
std::cout << "var的地址: " << &var << std::endl;
std::cout << "ptr的值: " << ptr << std::endl;
std::cout << "ptr指向的值: " << *ptr << std::endl;

指针运算

int arr[5] = {10, 20, 30, 40, 50};
int *ptr = arr;  // ptr指向arr的第一个元素

std::cout << *ptr << std::endl;  // 输出10
ptr++;  // 移动到下一个元素
std::cout << *ptr << std::endl;  // 输出20

动态内存分配

// 使用new分配内存
int *ptr = new int;   // 分配一个整数的内存
*ptr = 100;           // 在分配的内存中存储值

// 使用delete释放内存
delete ptr;

// 分配数组
int *arr = new int[10];  // 分配10个整数的数组

// 释放数组
delete[] arr;

结构体

结构体是一种用户自定义的数据类型，允许将不同类型的数据组合在一起。

结构体定义和使用

// 定义结构体
struct Person {
    std::string name;
    int age;
    float height;
};

// 创建结构体变量
Person person1;
person1.name = "张三";
person1.age = 25;
person1.height = 175.5f;

// 初始化结构体
Person person2 = {"李四", 30, 180.0f};

// 结构体指针
Person *ptr = &person1;
std::cout << ptr->name << std::endl;  // 使用->访问成员

结构体与函数

// 结构体作为函数参数
void printPerson(Person p) {
    std::cout << "姓名: " << p.name << std::endl;
    std::cout << "年龄: " << p.age << std::endl;
    std::cout << "身高: " << p.height << std::endl;
}

// 结构体作为函数返回值
Person createPerson(std::string name, int age, float height) {
    Person p;
    p.name = name;
    p.age = age;
    p.height = height;
    return p;
}
