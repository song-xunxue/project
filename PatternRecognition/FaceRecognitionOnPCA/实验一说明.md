# 实验一：基于PCA子空间分析的人脸身份识别

## 一、实验目的

1. 理解主成分分析（PCA）的数学原理，掌握特征提取和降维的方法
2. 实现基于PCA的人脸识别系统，在ORL人脸库上验证识别效果
3. 分析PCA降维维度（K值）对识别准确率的影响
4. 对比不同分类方法（欧氏距离1-NN、余弦相似度1-NN、SVM支持向量机）的优劣

## 二、实验环境

- **操作系统**: Windows 11
- **开发语言**: Python 3
- **核心依赖**: PyTorch（张量运算）、OpenCV（图像读取）、scikit-learn（SVM）、NumPy、Matplotlib
- **计算设备**: CUDA GPU / CPU
- **开发工具**: PyCharm

## 三、数据集

### ORL人脸数据库（AT&T Laboratories Cambridge）

- **来源**: AT&T Laboratories Cambridge，通过Kaggle下载
- **下载方式**: 使用 `kagglehub` 库从 Kaggle 平台下载（数据集标识: `sarfarazansari/att-database-of-faces`）
- **数据规模**: 40人，每人10张不同姿态和表情的灰度图像，共400张
- **图像格式**: PGM格式，分辨率 92×112 像素
- **目录结构**: `data/s1/1.pgm ~ 10.pgm`，`s1`~`s40` 对应40个不同身份

## 四、算法原理

### 4.1 PCA（主成分分析）降维

PCA的核心思想是将高维数据投影到方差最大的方向上，用少量主成分保留原始数据的主要信息。

**数学推导：**

设训练集有N个样本，每个样本展平后为D维向量（D = 92×112 = 10304）：

1. **计算平均脸**：
   $$\mu = \frac{1}{N}\sum_{i=1}^{N} x_i$$

2. **去均值**：
   $$\hat{x}_i = x_i - \mu$$

3. **构造协方差矩阵**（采用N×N技巧）：
   $$L = \hat{X} \cdot \hat{X}^T \quad (N \times N)$$

   > 当 N << D 时（本实验 N=320, D=10304），计算 N×N 矩阵的特征分解比 D×D 矩阵高效得多。

4. **特征分解**：
   $$L v_i = \lambda_i v_i$$

5. **选取前K个主成分**（按特征值降序排列）：
   $$u_i = \hat{X}^T v_i$$

6. **L2归一化**特征向量

7. **降维投影**：
   $$y = W^T (\hat{x})$$

   其中 W 为特征脸矩阵，形状 (D, K)。

### 4.2 三种分类方法

#### 方法一：欧氏距离 1-NN
$$d(x, y) = \|x - y\|_2 = \sqrt{\sum_{i}(x_i - y_i)^2}$$
取距离最近的训练样本标签作为预测。

#### 方法二：余弦相似度 1-NN
$$\cos(\theta) = \frac{x \cdot y}{\|x\| \cdot \|y\|}$$
取相似度最大的训练样本标签作为预测。余弦相似度关注向量方向的夹角，不受向量幅度影响。

#### 方法三：PCA + SVM
在PCA降维后的低维特征空间上训练SVM线性分类器：
- 核函数：线性核（linear kernel）
- 正则化参数：C = 1.0
- SVM寻找类别间的最优超平面决策边界

| 错误样本 | 欧氏距离 | 余弦相似度 | SVM |

- SVM在4个困难样本中正确识别了2个（#10和#45），体现了分类器学习决策边界的优势
- 欧氏距离和余弦相似度的总体准确率相同，但在个别样本上错误方向不同

## 七、分析与讨论

### 7.1 降维维度（K值）的影响

1. **信息保留 vs. 维度压缩**：K值越大，保留的方差信息越多，但计算成本也越高。实验表明，K=30~50已能保留足够的人脸判别信息。
2. **过低维度**：K=2或3时，大部分判别信息被丢弃，不同人的投影严重重叠，准确率很低。
3. **过高维度**：当K超过一定值后，新增的主成分主要是噪声，准确率不再提升甚至可能下降（过拟合风险）。

### 7.2 分类方法对比

1. **欧氏距离 1-NN**：
   - 优点：实现简单，无需额外训练
   - 缺点：对光照和幅度变化敏感，受噪声样本影响
   - 原理：基于绝对距离度量

2. **余弦相似度 1-NN**：
   - 优点：关注方向而非幅度，对光照变化更鲁棒
   - 缺点：同样受噪声样本影响（1-NN固有问题）
   - 原理：基于向量夹角度量

3. **PCA + SVM**：
   - 优点：学习全局最优决策边界，泛化能力更强，本实验中准确率最高（97.50%）
   - 缺点：需要额外的训练步骤，在高维空间下训练时间增加
   - 原理：在PCA子空间中寻找最大间隔超平面

### 7.3 实验结论

1. PCA能有效将10304维的人脸图像压缩到50维左右，同时保留足够的身份判别信息
2. 降维维度对识别效果有显著影响，存在最优K值范围
3. SVM分类器在PCA降维后的特征空间上表现优于简单的最近邻方法
4. PCA的N×N协方差技巧使得计算效率大幅提升（320×320 vs 10304×10304）
