# 实验三：基于LeNet-5卷积神经网络的Fashion-MNIST分类

## 一、实验目的

1. 理解卷积神经网络（CNN）的基本原理，掌握卷积、池化、全连接等核心操作
2. 实现经典 LeNet-5 网络架构，在 Fashion-MNIST 数据集上验证分类效果
3. 分析各类别的分类性能差异，理解模型在不同类别上的表现
4. 通过特征图可视化理解卷积层的工作机制

## 四、算法原理

### 4.1 LeNet-5 网络架构

LeNet-5 是 Yann LeCun 于1998年提出的经典CNN，原始设计用于手写数字识别。本实验适配 Fashion-MNIST（输入28×28），架构如下：

```
输入 (1×28×28)
    ↓
C1: Conv2d(1→6, 5×5, padding=2) + Sigmoid → (6×28×28)
    ↓
S3: AvgPool2d(2×2, stride=2) → (6×14×14)
    ↓
C4: Conv2d(6→16, 5×5) + Sigmoid → (16×10×10)
    ↓
S5: AvgPool2d(2×2, stride=2) → (16×5×5)
    ↓
Flatten → (400)
    ↓
F6: Linear(400→120)
    ↓
F7: Linear(120→84)
    ↓
F8: Linear(84→10) → 10类输出
```

### 4.2 关键组件

**卷积层（Conv2d）**：通过卷积核在输入特征图上滑动，提取局部空间特征。每层卷积核学习不同的特征模式（边缘、纹理等）。

**Sigmoid 激活函数**：原始 LeNet 使用 Sigmoid 而非 ReLU：
$$\sigma(x) = \frac{1}{1 + e^{-x}}$$

**平均池化（AvgPool2d）**：对局部区域取平均值，降低特征图分辨率，减少参数量。

**全连接层（Linear）**：将高维特征映射到10个类别的得分。

### 4.3 训练策略

- **损失函数**: CrossEntropyLoss（交叉熵）
- **优化器**: Adam（自适应学习率，lr=0.001）
- **批量大小**: 64
- **训练轮次**: 20 epochs
- **最优模型保存**: 验证集准确率最高时保存模型权重

### 6.4 混淆矩阵分析

> 见 `confusion_matrix.png`

主要混淆对：
- **Shirt ↔ T-shirt/top**：衬衫和T恤外形相似，是最大的混淆来源
- **Coat ↔ Pullover**：外套和套头衫轮廓接近
- **Shirt ↔ Coat**：长袖衬衫与轻薄外套难以区分

### 6.5 错误样本分析

> 见 `misclassified_samples.png`

共 1366 个错误样本。观察发现：
- 错误多集中在上装类（T-shirt、Shirt、Pullover、Coat）之间的混淆
- 鞋类（Sandal、Sneaker、Ankle boot）和包（Bag）几乎不会与服装混淆
- 一些错误样本即使人眼也难以区分

### 6.6 特征图可视化

> 见 `feature_maps.png`

展示了 C1 卷积层（6个通道）和 S3 池化层（6个通道）对输入图像的响应：
- 不同通道关注不同的特征模式（边缘、纹理、区域亮度）
- 池化层在保留主要特征的同时降低了空间分辨率

## 七、分析与讨论

### 7.1 类别间差异分析

准确率呈现明显的两极分化：
- **高准确率类别（>94%）**：Trouser、Sandal、Sneaker、Bag、Ankle boot — 这些类别在形状、纹理上与其他类别差异显著
- **低准确率类别（<85%）**：T-shirt、Pullover、Coat、Shirt — 这些上装在形状上高度相似，仅凭像素级特征难以区分

Shirt 类别准确率最低（57.90%），因为它在视觉上与 T-shirt、Coat、Pullover 都有重叠。

### 7.2 LeNet-5 架构的局限性

1. **Sigmoid 激活函数**：现代CNN普遍使用 ReLU，Sigmoid 在深层网络中容易梯度消失
2. **平均池化**：最大池化（MaxPool）通常能保留更强的特征响应
3. **网络深度浅**：仅2个卷积层，特征提取能力有限
4. **无正则化**：缺少 Dropout、BatchNorm 等技术

### 7.3 可能的改进方向

1. 使用 ReLU 替换 Sigmoid
2. 增加 BatchNorm 层
3. 使用更深的网络（如 VGG、ResNet）
4. 数据增强（随机翻转、旋转、缩放）
5. 学习率调度（如 ReduceLROnPlateau）

### 7.4 实验结论

1. LeNet-5 在 Fashion-MNIST 上达到 86.34% 的总体准确率，证明了 CNN 在图像分类任务上的有效性
2. 模型在形状差异显著的类别上表现优秀（>94%），在视觉相似的类别上表现较弱（<76%）
3. 特征图可视化直观展示了卷积层的特征提取过程
4. 经典 LeNet-5 架构虽有局限性，但作为入门CNN实验具有很好的教学价值
